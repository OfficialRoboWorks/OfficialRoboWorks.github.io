SMZ-S1L Classic Roofless Package

RoboWorks SMZ-S1L Classic Roofless Package is a content-pack for Flans mod which adds 50 new vehicles to the game. SMZs were tiny welfare cars produced in the Soviet Union. This package includes SMZ-S1L without an openable roof.

Installation

1. Download and install Forge for Minecraft 1.7.10
2. Download Flans mod for 1.7.10 and place into "mods" folder
3. Create "flan" folder in ".minecraft" directory
4. Download RoboWorks SMZ-S1L Classic Roofless Package and place it in "flan" folder
