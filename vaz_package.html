RoboWorks VAZ Package
(Legacy)

RoboWorks VAZ Package is a content-package for Flans mod which adds 21 legendary LADAs to the game. All models are highly detailed and well textured.

Features

• High detail
• Minecraft version 1.7.10
• 16x16 styled textures
• One driver and three passenger seats

Installation

1. Download and install Forge for Minecraft 1.7.10
2. Download Flans mod for 1.7.10 and place into "mods" folder
3. Create "flan" folder in ".minecraft" directory  
4. Download RoboWorks VAZ Package and place it in "flan" folder
