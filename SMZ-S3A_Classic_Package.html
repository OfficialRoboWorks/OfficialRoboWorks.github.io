SMZ-S3A Classic Package

RoboWorks SMZ-S3A Classic Package is a content-pack for Flans mod which adds 24 new vehicles to the game. SMZs were tiny welfare cars produced in the Soviet Union. This package includes SMZ-S3A with an openable roof.

Features

• High detail
• Minecraft 1.7.10
• 16x16 styled textures
• Realistic engine sound
• One driver and one passenger seat
• Roof can be opened by pressing "K"

Installation

1. Download and install Forge for Minecraft 1.7.10
2. Download Flans mod for 1.7.10 and place into "mods" folder
3. Create "flan" folder in ".minecraft" directory
4. Download RoboWorks SMZ-S3A Classic Package and place it in "flan" folder
